# Практическая работа №1. Основные типы и конструкции в C++

## Требования к программе

*   Программа должна состоять из **одного файла с кодом** (`*.cpp`).
*   Файл должен содержать:
    *   Основную функцию `main` с консольным меню.
    *   Дополнительные функции.
*   **Структура файла:**
    1.  Предварительные объявления дополнительных функций.
    2.  Определение функции `main`.
    3.  Определения дополнительных функций (после `main`).
*   **Ввод данных от пользователя** осуществляется **только** в функции `main`.
*   **Вывод результатов** всех функций (кроме `showTypeInfo`) также выполняется в `main`.

## Дополнительные функции

1.  **`belongsToInterval`**
    *   **Назначение:** Определяет, принадлежит ли значение `x` заданному интервалу.
    *   **Параметры (порядок):** `x`, `bottom`, `top`.
    *   **Возвращает:** Результат проверки.

2.  **`minValue`** (для целых чисел)
    *   **Назначение:** Возвращает меньшее значение из двух **целочисленных** аргументов.
    *   **Требование:** В реализации используйте **тернарный оператор**.

3.  **`minValue`** (для вещественных чисел)
    *   **Назначение:** Возвращает меньшее значение из двух аргументов типа **`float`**.
    *   **Примечание:** Функции 2 и 3 являются перегруженными (различаются типами аргументов).

4.  **`order`**
    *   **Назначение:** Возвращает количество знаков целочисленного аргумента **без учета знака минус**.
    *   **Пример:** Для числа `-745` возвращает `3`.

5.  **`showTypeInfo`**
    *   **Назначение:** Выводит на консоль информацию о типах: `char`, `int`, `short`, `long`, `long long`.
    *   **Информация:** размер (через `sizeof`), максимальное и минимальное значение.
    *   **Библиотека:** Используйте константы из `<climits>` (`<limits.h>`).
    *   **Особенность:** Вывод выполняется внутри этой функции (не в `main`).

6.  **`getLetterCode`**
    *   **Назначение:** Возвращает числовое представление (код) символа.

7.  **`isPrime`**
    *   **Назначение:** Проверяет, является ли аргумент простым числом.

## Работа с перегруженными функциями

*   Функции `minValue` (2 и 3) различаются типами аргументов.
*   В зависимости от ввода пользователя (целые или вещественные числа) должна вызываться **соответствующая** версия функции.

## Реализация меню в `main`

*   Необходимо реализовать **консольное меню** для выбора и проверки функций.
*   Организуйте с помощью цикла `while` и конструкции `switch` для ветвления.
*   **Обязательна** обработка некорректного пользовательского ввода.
